# MegaBook Versión Web 📚

**MegaBook** es una aplicación diseñada para transformar la lectura en un hábito imparable. Utiliza principios de psicología del comportamiento y gamificación para ayudarte a mantener la constancia, registrar tus libros favoritos y medir tu progreso diario.

Esta versión web es el PMV en producción.

---

## 🔥 Características Destacadas

*   **Rachas Infinitas (Success Bridge)**: Un sistema de streaks que no se rompe al cambiar de semana.
*   **Temporizador de Enfoque**: Sesiones de lectura controladas para maximizar tu concentración.
*   **Integración con Google Books**: Busca y agrega libros reales a tu biblioteca personal en segundos.
*   **Habit Stacking**: Configura tus sesiones basándote en anclajes de hábitos preexistentes.
*   **Persistencia Total**: Gracias al almacenamiento local enriquecido, tu progreso está siempre seguro en tu navegador.

## 🛠️ Tecnologías

*   **React** (Vite)
*   **Context API** para la gestión de estado global.
*   **Custom Hooks** (`useLocalStorage`) para persistencia transparente.
*   **TailwindCSS** para un diseño moderno y oscuro.
*   **Formik** para validación de formularios robusta.
